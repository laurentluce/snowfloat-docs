Install guide¶

SnowFloat client depends on python-requests:

$ pip install requests

Next, get the latest SnowFloat client code from github and install it:

$ git clone git://github.com/snowfloat/snowfloat-python.git
$ cd snowfloat-python
$ sudo python setup.py install

To verify that SnowFloat can be seen by Python, type python from your shell. Then at the Python prompt, try to import SnowFloat:

>>> import snowfloat

Optional: Install Shapely to have the local geometries also be Shapely geometries.

That’s it – you can now move onto the tutorial.
